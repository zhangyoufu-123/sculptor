# Sculptor v7.0: Thinking Workspace — 实施计划

> 基于 v7.0-Architect 方案，纯增量改进，完全兼容 v6.0

## P0: 核心转向（今天）

### 1. Ghost Text 触发优化
- [ ] 旧：停止输入 400ms 触发
- [ ] 新：光标停留 800ms 或 句号/问号/感叹号后 400ms
- [ ] 连续 3 次拒绝 → 本次会话不再触发
- [ ] 默认单候选，Alt+Tab 查看其他

### 2. AI 不再聊天
- [ ] ChatPanel 对话模式改为"动作+结果"
- [ ] AI 响应格式：执行 → 更新 Outline → 简短确认
- [ ] 删除冗长 AI 对话历史
- [ ] 保留架构引导提问（最多 2 轮）

### 3. Outline 始终可见
- [ ] 写作页左侧结构树高亮当前节点（静态金色边框）
- [ ] AI 内容自动归属到对应节点
- [ ] 节点右键：AI展开/AI填充/添加笔记/添加参考

## P1: 工作区重组

### 4. Studio（原 EchoWall）
- [ ] 重命名 EchoWall → Studio
- [ ] Tab 结构：Notes → References → Versions → AI
- [ ] AI Tab 默认隐藏，主动召唤显示，30s 自动关闭

### 5. Author Memory
- [ ] author_memory 表 + API
- [ ] 偏好/厌恶/习惯/代表作/禁用表达
- [ ] System Prompt 注入 Author Memory
- [ ] 禁用表达后处理过滤

### 6. Universal Command Palette
- [ ] Cmd+K 升级为 /命令 模式
- [ ] /rewrite /continue /expand /outline /research /style /compare /summarize /health /focus /draft

### 7. Drafts 管理
- [ ] Cmd+Shift+D 创建 Draft
- [ ] Draft 列表 + 对比 + AI 合并
