# Sculptor v8.0 — 全文本覆盖方案

## 新增体裁

在现有五文体基础上扩展为 **12 种体裁**：

| 体裁 | 触发词 | 节点结构 | 优先数据源 |
|------|--------|----------|-----------|
| 议论文 | 论证/论/反驳 | thesis→argument→counter→evidence→rebuttal→conclusion | 现有 |
| 记叙文 | 故事/经历/回忆 | hook→scene→scene→climax→reflection | 现有 |
| 说明文 | 解释/说明/为什么 | hook→definition→component→step→summary | 现有 |
| 散文 | 散文/随笔/感悟 | hook→imagery→imagery→reflection | 现有 |
| 游记 | 游记/旅行/之旅 | hook→departure→scene→scene→impression→reflection | 现有 |
| **论文** | 论文/学术/研究 | abstract→intro→literature→method→result→discussion→conclusion | funNLP学术语料 |
| **朋友圈** | 朋友圈/配文/短文案 | hook→emotion→punchline | 社交媒体语料 |
| **视频文案** | 视频/脚本/抖音/B站 | hook→setup→body→climax→CTA | 短视频语料 |
| **戏剧** | 戏剧/剧本/话剧 | act→scene→dialogue→stage_direction→climax | 戏剧语料 |
| **诗歌** | 诗词/诗歌/写诗 | title→line1→line2→...→couplet | 唐诗宋词库 |
| **演讲稿** | 演讲/发言/致辞 | opening→body1→body2→body3→closing | 演讲语料 |
| **商业文案** | 广告/推广/品牌/文案 | headline→subhead→body→features→CTA | 广告语料 |

## 实施路径

### Phase A: 体裁引擎扩展（1天）
- `types/genre.ts`：新增 Genre 类型（12 种）
- `lib/ai/genre-detector.ts`：关键词 + 规则检测
- `app/api/architect/chat/route.ts`：扩展 getDeepQuestion 覆盖 12 体裁

### Phase B: funNLP 语料集成（1天）
- `lib/corpus/`：语料检索模块
- 本地内置：诗词库、名言库、成语库
- funNLP 参考：词库、文本纠错、文本生成
- 后续可扩展：AI 生成时从语料库检索匹配素材

### Phase C: 节点模板库（1天）
- `lib/templates/`：12 种体裁的节点模板
- 每种体裁预设 2-3 种变体（如论文：学术型/科普型）
